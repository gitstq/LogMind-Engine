# 更新日志

所有重要的项目变更都将记录在此文件中。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/) 规范。

## [1.0.0] - 2026-05-12

### 🎉 首次发布

#### 新增功能

- ✅ **多格式日志解析**
  - 支持 JSON 格式日志
  - 支持纯文本格式日志
  - 支持 Apache/Nginx 访问日志
  - 支持 Syslog 系统日志
  - 自动格式检测

- ✅ **智能错误分析**
  - 错误自动分组
  - 模式提取与归一化
  - 错误频率统计
  - 时间分布分析

- ✅ **异常检测**
  - 错误率监控
  - 错误突增检测
  - 综合异常分数

- ✅ **TUI 用户界面**
  - 基于 Rich 库的美观界面
  - 彩色日志级别显示
  - 交互式操作模式

- ✅ **AI 增强功能**
  - OpenAI GPT 支持
  - Anthropic Claude 支持
  - Ollama 本地模型支持
  - 智能修复建议
  - 响应缓存

- ✅ **CLI 命令行工具**
  - `analyze` - 分析日志文件
  - `stats` - 显示统计信息
  - `errors` - 显示错误分组
  - `entries` - 显示日志条目
  - `report` - 生成完整报告
  - `interactive` - 交互式模式

- ✅ **Python API**
  - 完整的模块化设计
  - 易于集成的 API
  - 丰富的文档

#### 技术特性

- 🐍 纯 Python 实现
- 📦 零依赖（核心功能）
- 🔧 跨平台支持
- 📝 完整测试覆盖
- 📚 多语言文档（中/英/繁/日）
